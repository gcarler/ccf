# Aesthetic Expert Skill

Este skill dota al agente de la capacidad de diseñar y refinar interfaces siguiendo el sistema de diseño "Clean Productivity" de CCF.

## When to use this skill
- Al crear nuevas páginas o módulos en el frontend.
- Para refinar componentes existentes que se sientan "MVP" o poco premium.
- Cuando se necesite asegurar la consistencia visual en todo el ecosistema.

## How to use it
1. **Consultar Guías:** Siempre referenciar `brain/ccf_design_guidelines.md` antes de proponer cambios visuales.
2. **Aplicar Layout:** Priorizar el sistema de doble barra lateral y navegación por breadcrumbs.
3. **Optimismo Visual:** Implementar feedback inmediato (ghosting en drag-and-drop, actualizaciones de contadores en tiempo real).
4. **IA en Todas Partes:** Diseñar puntos de entrada para IA (shimmer effects, contextual triggers) que no interrumpan el flujo principal.
5. **Jerarquías Claras:** Usar anidamiento visual y badges de relación para clarificar dependencias complejas.
